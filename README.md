# Student Grade Management System

A simple C++ application for managing student grades. This project demonstrates basic C++ concepts and is perfect for learning version control with Git.

## Features

- Add new students
- Add grades to students
- View individual student information
- View all students and their grades
- Calculate average grades automatically

## Building the Project

### Prerequisites

- CMake (version 3.10 or higher)
- C++ compiler with C++17 support
- Git (for version control)

### Build Instructions

1. Clone the repository:
```bash
git clone <repository-url>
cd student_grade_system
```

2. Create a build directory and navigate to it:
```bash
mkdir build
cd build
```

3. Generate build files:
```bash
cmake ..
```

4. Build the project:
```bash
cmake --build .
```

## Running the Application

After building, run the executable:
```bash
./grade_system
```
